# 🤖 AgentHire AI

**Multi-Agent Interview Intelligence | Computer Vision | Generative AI Assessment Platform**

AgentHire AI is an AI-powered interview assessment platform that simulates real-time technical and HR interviews using multiple AI agents. It evaluates candidates through Computer Vision, Large Language Models (LLMs), and Generative AI to provide intelligent feedback, performance analytics, and personalized improvement recommendations.

---

## 🚀 Features

### 🎙️ Multi-Agent Interview Simulation
- AI Interviewer Agent conducts technical and behavioral interviews.
- Dynamic follow-up questions based on candidate responses.
- Realistic conversational interview experience.

### 🧠 Generative AI Evaluation
- Evaluates candidate responses using LLMs.
- Measures:
  - Technical knowledge
  - Communication skills
  - Confidence
  - Problem-solving ability
  - Overall interview performance

### 👁️ Computer Vision Assessment
- Real-time webcam analysis using OpenCV.
- Detects:
  - Eye contact
  - Face presence
  - Head posture
  - Attention level
  - Basic engagement metrics

### 📊 Performance Analytics
- Interview score dashboard
- Skill-wise evaluation
- Response quality analysis
- Communication insights
- Strengths & improvement areas

### 📝 Personalized Feedback Report
- AI-generated interview summary
- Detailed strengths
- Weakness identification
- Actionable recommendations
- Learning roadmap for improvement

### 💾 Data Management
- Stores interview history
- Candidate performance tracking
- SQLite database integration

# ⚙️ How It Works

1. Candidate starts the interview.
2. AI Interview Agent asks dynamic questions.
3. Candidate responds through text or voice.
4. Computer Vision continuously analyzes facial engagement.
5. LLM evaluates response quality.
6. Multiple AI agents collaborate to assess overall performance.
7. Candidate receives a comprehensive feedback report with analytics.

# 🖥️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AgentHire-AI.git
```

Navigate to the project

```bash
cd AgentHire-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
